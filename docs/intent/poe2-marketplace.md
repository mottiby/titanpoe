# Интент: Маркетплейс услуг по Path of Exile 2 (по образцу overgear)

> Подтверждено через скилл `interview-me` 2026-06-24. Явное «да» получено.

## Подтверждённый интент

- **Итог (текущий деливерабл):** Пакет подготовки из 4 документов —
  1. Разбор overgear (teardown)
  2. ТЗ/спецификация продукта (PRD)
  3. Техническая архитектура (под кодогенерацию)
  4. Роадмап MVP → полная версия

  **Кода на этом этапе нет.**
- **Продукт:** Курируемый маркетплейс услуг **только по PoE2**, покрывающий **весь спектр** услуг как у overgear: валюта, предметы/гир, прокачка, карри боссов, фарм Атласа, прохождение челленджей, крафт, коучинг, продажа аккаунтов.
- **Модель:** Курируемый / platform-mediated маркетплейс (как overgear) — владелец верифицирует продавцов-исполнителей, платформа держит средства в **эскроу**, даёт гарантию, разрешает споры и берёт **комиссию**. НЕ открытая немодерируемая доска объявлений.
- **Владелец/исполнитель:** Соло-разработчик, строит сам с помощью ИИ (Claude). Сначала документы, потом код.
- **Предлагаемый стек (уточняется в доке «архитектура»):** Next.js + TypeScript + PostgreSQL + Prisma + Stripe Connect (эскроу/выплаты).
- **Критерий успеха:** Документы настолько конкретны, что можно сразу начинать кодить MVP без додумывания.
- **Связывающее ограничение:** Всё должно быть посильно **одному** разработчику с ИИ — сложность держим управляемой.
- **Вне охвата (сейчас):** сам код/реализация, визуальный брендинг, юридическое/бухгалтерское оформление компании (риски и ToS — отдельным разделом, но это не юрконсультация).
